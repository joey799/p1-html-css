Antwoorden op Algemene Vragen

Waarom noem je de eerste HTML-pagina; index.html?

De eerste HTML-pagina wordt vaak "index.html" genoemd omdat veel servers deze pagina automatisch herkennen als de standaardpagina van een website. Wanneer een gebruiker een domein bezoekt zonder een specifiek bestand in de URL, toont de server standaard het bestand genaamd "index.html".

Met welke HTML versie werken we nu?

De huidige versie van HTML is HTML5. Deze versie is sinds oktober 2014 de officiële standaard en bevat verbeteringen voor semantiek, multimedia-elementen, en mobiele optimalisatie.

Met welke CSS versie werken we nu?

De huidige versie van CSS is CSS3, maar CSS wordt continu doorontwikkeld via modules. CSS3 zelf wordt sinds 1999 verbeterd, maar veel modules zoals Flexbox, Grid, en andere zijn de afgelopen jaren toegevoegd of verbeterd.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser?

Elke browser interpreteert en rendert HTML, CSS en JavaScript net iets anders. Dit komt door verschillende engines die browsers gebruiken (zoals Blink in Chrome en Edge, Gecko in Firefox). Deze engines hebben subtiele verschillen in hoe ze webstandaarden toepassen, wat ervoor kan zorgen dat websites er in elke browser net iets anders uitzien.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populairste browser wereldwijd is Google Chrome. Chrome gebruikt vaak meer RAM dan andere browsers vanwege zijn manier van werken met tabs en processen. Het RAM-gebruik van Chrome kan variëren afhankelijk van het aantal geopende tabbladen, maar het gebruikt meestal tussen de 200 MB en 1 GB RAM per tabblad, afhankelijk van de complexiteit van de pagina's.
